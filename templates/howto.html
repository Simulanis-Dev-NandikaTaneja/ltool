Language Translator

The Language Translator Expects the file to be either:

      .xlsx
      .txt
      .csv

if you upload a file of any other type the webpage display's the message File type not supported

if you're uploading a csv or xlsx file please make sure the first column contains the text to be tranlated(i.e. contains the text written in English)

if the language translator is unable to read the contents of the uploaded file the webpage display's the message File not readable......in case of such a message please make sure the columns of your file are well seperated
